## Features

### User-Friendly Station Selection
- **Select Any Station**: Users can easily select from a list of available metro stations.

### Nearest Station Locator
- **Current Location**: The app shows the nearest station based on the user's current location.
- **Destination-Based Stations**: Displays the closest stations along the route to the selected destination.

### Route Management
- **Save Routes**: Users can save their frequently used routes.
- **Route Details**: Detailed view of saved routes, including time, pricing, and additional information.

### Multilingual Support
- **Language Options**: The app supports both Arabic and English languages for a wider user base.

### Pricing Information
- **Metro Ticket Costs**: Provides users with information on metro ticket pricing.
- **Cost Calculation**: Automatically calculates the cost of the metro fare based on the selected route.

### Metro Operation Schedule
- **Schedule Access**: Users can view the operational schedule for all metro lines.

### Map Integration
- **Map Display**: Visual representation of the nearest station according to the user's location on the map.

## Dependencies

### Development and Linting
- **flutter_lints**: Improves code quality and adherence to best practices.
- **dartx**: Provides additional functionalities for the Dart programming language.

### UI and User Interaction
- **drop_down_list**: Implements drop-down menus for user selection.
- **fluttertoast**: Displays short notifications on the screen.
- **material_floating_search_bar_2**: Offers a user-friendly search bar component.

### Location and Maps
- **geolocator**: Determines the user's location for various purposes.
- **geocoding**: Translates addresses to geographical coordinates and vice versa.
- **google_maps_flutter**: Integrates Google Maps functionality into your app.
- **flutter_polyline_points**: Simplifies drawing polylines (lines connecting points) on maps.

### State Management and Dependency Injection
- **dartz**: Provides functional programming tools for handling data types and errors.
- **flutter_bloc**: Implements the BLoC (Business Logic Component) state management pattern.
- **get_it**: Offers a simple service locator for dependency injection.
- **get**: Additional state management and dependency management library.

### Networking and Data Persistence
- **dio**: Makes efficient HTTP requests for fetching data from APIs.
- **shared_preferences**: Stores simple key-value pairs for persistent data.
- **sqflite**: Enables working with SQLite databases for more complex data storage.
- **path_provider**: Provides access to application-specific directories for data storage.

### Additional Dependencies
- **url_launcher**: Opens external links in a user's default browser.
- **uuid**: Generates unique identifiers.
- **workmanager**: Schedules background tasks for periodic or delayed execution.
- **google_fonts**: Allows using Google Fonts for more customization.
- **font_awesome_flutter**: Provides the Font Awesome icon library for visual elements.
